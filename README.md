#Project 8 - Mobile-Recharge-Request-System

Developed a ServiceNow Mobile Recharge Request System using Service Catalog and Flow Designer to automate approvals and request fulfillment.

Project Overview:
This solution streamlines the process of requesting mobile recharges through a Service Catalog item, ensuring automated approvals, efficient processing, and timely notifications.

Key Features:
- User-friendly Catalog Item for recharge requests
- Dynamic form with validation (Mobile Number, Operator, Plan, Amount)
- Approval workflow using Flow Designer
- Automated task creation for fulfillment
- Real-time email notifications for request status updates
- End-to-end process automation

Technologies Used:
- ServiceNow ITSM
- Service Catalog
- Flow Designer
- Notifications

Outcome:
- Reduced manual effort in handling recharge requests
- Improved request tracking and transparency
- Faster processing with automation
